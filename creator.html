To avoid having to find a single time slot for everyone, this year you can start the contest when it suits you the best.

Creating a contest
1. Go to the contest creator page: Contest Creator Page
2. Select the contest you want to create.
3. Enter your domjudge username and password.
The only way to register is by sending me an e-mail as described in registration.
This will create a copy of the contest just for you starting one minute after the creation and lasting 2h30m. Only at the end of the global timeslot of the contest you will be able to see the scores of the other participants in the results page.
You can only create the contest once so be sure to be ready when you do it.
4. Go to domjudge and login and solve the problems. You might have to wait some time (less than 1 minute) before you can see your contest on domjudge. Be patient.
